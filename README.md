# OS Agent

Natural language interface for executing operating system commands, automating file operations, and managing system processes through conversational input.

## Features

- Natural language to shell command translation
- File and directory management via conversation
- Process monitoring and management
- System information queries
- Command confirmation for destructive operations
- Execution history and undo support

## Tech Stack

Python, LLM APIs, Subprocess, OS Libraries

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/OS-Agent.git
cd OS-Agent
pip install -r requirements.txt
```

### Usage

```bash
python agent.py
```
